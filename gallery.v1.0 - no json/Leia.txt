[1] Se não houver nenhuma galeria requisitada na URL, uma galeria padrão é carregada, como pode ver no seu 
    primeiro acesso.

--------------------------------------------------------------------------------------------------

[2] Este script,  plugin,  site, como quiser chamar...  identifica automaticamente as URLs atuais,  ou seja, 
    você pode executar um teste completo em localhost e depois simplismente copiar para um servidor externo, 
    as URLs das galerias são atualizadas.

--------------------------------------------------------------------------------------------------

[3] Não armazena imagens.

--------------------------------------------------------------------------------------------------

[4] QUANTIDADE DE CARACTERES NA URL: Por padrão as primeiras galerias contem apenas um caractere.
    Ex: http://site/gallery/i?=A, o número de caracteres aumentará de forma automática, gradual e 
    cada vez mais lento, pois é preciso que o nome de uma galeria já esteja no registro. Ex:
    gallery/i?=A ... gallery/i?=Ad ... gallery/i?=Ad7 .... gallery/i?=7AdsYg5h17k

    AUMENTAR O NÚMERO DE CARACTERES: Não quer esperar o número aumentar? => veja o arquivo "gallery/docs/limite.txt" 
    simples assim. Coloque o número que quiser.

--------------------------------------------------------------------------------------------------

[5] SENHA:  Primeito  acesso na galeria você é consultado  sobre criar uma senha para area administrativa, 
    crie uma, teste o login. Depois será consultado sobre remover o link para sua tabela de administração. 
    Mesmo trancada com senha, recomendamos remove-lo.
 
    SHA256, é uma criptografia sem descriptografia,  as vezes chega a ser um problema, pois se você perder 
    uma senha dessas, fudeu. Mas, é ela mesma.

--------------------------------------------------------------------------------------------------
[6] REMOVER LIKS do final da página: vá ao diretório "gallery/docs/links/" e delete o arquivo .txt 
    correspondentes ao menu que você quer remover.
